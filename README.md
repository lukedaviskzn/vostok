# Vostok - A Gemini Browser

A permanently incomplete browser for the [Gemini Protocol](https://geminiprotocol.net/) a minimalist alternative to HTTP.

Features:
* Text
* Links
* Tabs (kind of)
* Tab History

Missing Features:
* User certificates
* User input
* ...
